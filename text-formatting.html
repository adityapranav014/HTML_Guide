Text Formatting

1. Bold Text

The <b> tag is used to make text bold. It is often used for emphasizing important content.

Example:

This is bold text

2. Italic Text

The <i> tag is used to make text italic. It is commonly used for quotes or foreign words.

Example:

This is italic text

3. Underlined Text

The <u> tag is used to underline text. It was originally used to indicate links, but now it is used for other purposes.

Example:

This is underlined text

4. Strikethrough Text

The <s> or <del> tag is used to apply a strikethrough effect on text.

Example:

This text is struck through

This text is also struck through

5. Superscript Text

The <sup> tag is used to define superscript text. This is often used in mathematical expressions or footnotes.

Example:

Water is H2O

6. Subscript Text

The <sub> tag is used to define subscript text. This is often used in chemical formulas.

Example:

CO2

7. Code Text

The <code> tag is used to display inline code or programming syntax.

Example:

Here is an example of inline code: let x = 10;

8. Small Text

The <small> tag is used to display text in a smaller font size. This is often used for copyright information or disclaimers.

Example:

This is small text

9. Marked Text

The <mark> tag is used to highlight text, usually for search results or important information.

Example:

This is highlighted text

10. Blockquote

The <blockquote> tag is used to define a block of text that is a quotation from another source.

The <pre> tag is used to display preformatted text, where the spacing and line breaks are preserved.
