1. Ordered List

An ordered list is used when the order of the list items matters. It is represented by the <ol> element.

Here is an example of an ordered list:

  1. First item
  2. Second item
  3. Third item

2. Unordered List

An unordered list is used when the order of the list items doesn't matter. It is represented by the <ul> element.

Here is an example of an unordered list:

  • Apple
  • Banana
  • Orange

3. Description List

A description list is used when you want to pair terms with their descriptions. It is represented by the <dl>, <dt>, and <dd> elements.

Here is an example of a description list:

HTML
A markup language used for creating web pages.
CSS
A stylesheet language used for describing the look of web pages.
