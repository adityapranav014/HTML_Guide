1. Basic Table

A table is used to represent data in a grid format. It is defined using the <table> element. The table consists of rows and columns.

Here is an example of a basic table:

Name Age City
John 25 New York
Alice 30 Los Angeles
Bob 28 Chicago

2. Table with Header

A table header is defined using the <th> element. It typically contains the column names or labels.

Here is an example of a table with a header:

Product Price Quantity
Apple $2 5
Banana $1.5 10
Orange $3 8

3. Table with Rowspan and Colspan

The rowspan and colspan attributes are used to merge cells across rows or columns.

Here is an example of a table with rowspan and colspan:

Name Scores
Math English
John 85 90
Alice 88 92
Bob 75 80

4. Table with Caption

A table caption can be added using the <caption> element. It provides a title or description for the table.

Here is an example of a table with a caption:

Employee Data
Employee ID Employee Name Department
1001 John Doe IT
1002 Jane Smith HR
1003 Mark Lee Finance
